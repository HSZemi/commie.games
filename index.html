FAQ

I am a complete noob, can I still participate?
Yes, of course! There is no rating prerequisite.
Do I need to download special mods to play?
No. The game will download special mods/maps automatically when you join the game.
If I get rigged in twice in a day, can I play in two games?
No, unless T90 says so.
Which game do I need?
Age of Empires II Definitive Edition
Do I need all the expansions to play?
No. The base game is sufficient.
Can I play with the Steam/Xbox/MS Store version of the game?
Steam works, MS Store works, Xbox only works if the game mode does not involve a data mod or custom map. This is a technical limitation of that game version.
Why can I not react to the post from Rig-O-Mat 97.1?
You probably joined the Discord less than 10 minutes ago. There is a 10-minute cooldown before you can chat or react.
Do I need Voobly?
No. That was a long time ago.
Is it rigged?
Most definitely.

Host

You got selected to host a community game? Here's what you need to do.

Step 1Subscribe to the Necessary Mods

Make sure you are subscribed to the mods that include the map script and possibly the data mod that shall be used for the game.

Some common mods: King Boom, Exploding Villagers, Zetnus HyperRandom, Zetnus Random Map Script Pack (includes Forest Nothing), Pilgrims Nothing DE, Random Cost Mod.

Step 2Create a Game

Make sure that Spectators are enabled and that, if necessary, the correct data mod is selected.

Choose a simple password that cannot be easily guessed.

Step 3Configure the Game

  • Select the map script. Maps from mods can be found under Map Type "Custom".
  • Select the map size. Usually Large (8 Players), unless specified otherwise.
  • Select the proper game mode.
  • Configure diplomacy options. A free-for-all ("FFA") game has the "Fixed Teams" checkbox checked, while a diplomacy ("diplo") game has it unchecked.

Step 4Send the Game Information to T90

Send the game ID, the password, and a screenshot of your game settings to T90 via Discord direct message. There is a button to copy the game ID at the top right, and you can take a screenshot by pressing Win+Shift+S.

T90 will share the game information with the selected players.

Step 5Wait For All the Players to Join

Wait for all players that have been selected to join the game. Make sure no two players pick the same colour, and have them select the correct team. Usually that means everybody selects no team (-).

Step 6Start the Game Already!

Have the players select "I'm Ready!" and click "Start Game". Have fun!
